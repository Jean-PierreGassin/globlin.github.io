Globlin AH Bot

Install Globlin in your Discord server, subscribe, and manage tracked auction items directly from slash commands.

Install on Discord How It Works

Core Features

Real-Time Auction Scans

Continuously checks Blizzard auction data and flags listings at or below your configured undercut threshold.

Per-Server Tracking

Each Discord guild has isolated tracked items and subscription status for true multi-tenant operation.

Item-Level Aware Pricing

Supports item-level-specific tracking for gear where value shifts heavily by ilvl.

Subscription Access

Subscribe from Discord and unlock tracked item management and alert delivery for your server.

Web Control Panel

Optional local panel to start or stop the bot process, inspect logs, and review tracked item summaries.

Install and Activate

  1. Install the app to your Discord server

    Use the install link and select the server where you want Globlin alerts: 

    https://discord.com/oauth2/authorize?client_id=1246720491486580796

  2. Run /setup in your server

    This posts onboarding instructions and a Subscribe button in the current channel.

  3. Subscribe

    Click the Subscribe button and complete checkout. Your server gets activated once payment succeeds.

  4. Verify access with /subscription_status

    Confirm your subscription status and alert channel are active.

  5. Manage tracked items with /items

    Add or remove items and choose standard market-average tracking or item-level-specific tracking.

Customer Commands

/setup Post onboarding and subscribe button.

/subscription_status Show guild subscription state.

/items Open guided tracked-item management panel.

How Alerts Work

  • Bot loads cached item + connected realm data at startup.
  • It polls auction houses on a recurring async loop.
  • Listings are compared to market averages or configured item-level price targets.
  • Matching deals trigger formatted Discord alerts in your configured channel so you can snipe quickly.

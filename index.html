Install and Activate

  1. Install the app to your Discord server

    Select the server where you want Globlin alerts:

    Install on Discord

  2. Run /setup in your server

    This posts onboarding instructions and a Subscribe button in the current channel.

  3. Subscribe

    Click the Subscribe button and complete checkout. Your server gets activated once payment succeeds.

  4. Verify access with /subscription_status

    Confirm your subscription status and alert channel are active.

  5. Manage tracked items with /items

    Add or remove items and choose standard market-average tracking or item-level-specific tracking.

Customer Commands

/setup Post onboarding and subscribe button.

/subscription_status Show guild subscription state.

/items Open guided tracked-item management panel.

How Alerts Work

  • Bot loads cached item + connected realm data at startup.
  • It polls auction houses on a recurring async loop.
  • Listings are compared to market averages or configured item-level price targets.
  • Matching deals trigger formatted Discord alerts in your configured channel so you can snipe quickly.

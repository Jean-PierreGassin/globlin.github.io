FAQ

What is Globlin?

Globlin is a Discord bot for World of Warcraft Auction House deal alerts. It helps players identify underpriced listings quickly.

Who should use this bot?

Players who buy and flip items for gold, guild economists, and anyone who wants faster visibility into WoW market opportunities.

How do I start?

Install Globlin to your Discord server, run /setup, start the 1-day trial if eligible, then configure tracked markets.

How does the free trial work?

Trials are bot-native (not Stripe trials) and last 24 hours from activation. A server can only use one lifetime trial, and a user can only start one lifetime trial.

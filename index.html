Speed Architecture That Converts to Better Snipes

  • @tasks.loop(seconds=1) keeps scan orchestration continuous.
  • Realm scans run with AUCTION_SCAN_CONCURRENCY=12.
  • Fast polling window checks realms every 2-3 seconds near expected AH update time.
  • Outside the fast window, realms are revisited roughly every 20-23 seconds with jitter to reduce burst traffic.
  • Blizzard request pacing is set to 75ms minimum between requests in current config.
